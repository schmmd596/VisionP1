⚡ DÉMARRAGE RAPIDE (5 minutes):
================================================================================

Étape 1: Sur votre VPS Linux (SSH)
---
curl -fsSL https://ollama.ai/install.sh | sh
sudo systemctl start ollama
sudo systemctl enable ollama
ollama pull mistral:latest

Étape 2: Configuration Dolibarr
---
1. Allez à: Configuration → Chatbot IA → Setup
2. Sélectionnez: [🆓 Ollama] Mistral 7B
3. URL Ollama: http://localhost:11434
4. Laissez la clé API VIDE
5. Cliquez Sauvegarder

Étape 3: Test
---
Cliquez "🔌 Tester la connexion API"
Doit répondre: ✅ Connexion réussie!

🆘 SI VOUS AVEZ DES PROBLÈMES:
================================================================================

1. Connexion refusée:
   sudo systemctl restart ollama
   sudo journalctl -u ollama -f

2. Out of memory:
   Utilisez TinyLlama au lieu de Mistral

3. Pas d'accès depuis Dolibarr distant:
   Vérifiez firewall, modifiez config Ollama pour --host 0.0.0.0:11434

4. Pour plus: Voir OLLAMA_INSTALLATION.md (section Troubleshooting)
